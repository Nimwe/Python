Utilisation

Créer une clé-valeur

"cle_1" : "Valeur_1"


Création d'un dictionnaire

nom_du_dictionnaire = {
"cle_1" : "Valeur_1",
"cle_2" : "Valeur_2",
"cle_3" : "Valeur_3",
"cle_4" : ["Valeur_4", "Valeur_5"],
}
>>> (En attente d'instructions )


Parcourir un dictionnaire

Méthode 1 :
for i in nom_du_dictionnaire.items() :
print (i)

Retourne => ('cle_1', "Valeur_1")
('cle_2', 'valeur_2')
('cle_3', 'valeur_3')
('cle_4', ['valeur_4", "Valeur_5'])

Méthode 2 :
for cle,valeur in nom_du_dictionnaire.items() :
print("L'élément de clé", cle, "vaut", valeur)

Retourne => L'élément de clé cle_1 vaut Valeur_1
L'élément de clé cle_2 vaut valeur_2
L'élément de clé cle_3 vaut valeur_3
L'élément de clé cle_4 vaut ['valeur_4", "valeur_5']


Accéder à une valeur via la clé :

nom_du_dictionnaire["cle_1"]

Retourne => "valeur_1"


Ajouter une clé-valeur :

* Si la clé n'éxiste pas, il suffit d'ajouter une nouvelle clé dans le dictionnaire existant :
nom_du_dictionnaire["cle_5"] = "valeur_6" for i in nom_du_dictionnaire.items() : print (i)

Retourne => ('cle_1', "valeur_1")
('cle_2', 'valeur_2')
('cle_3', 'valeur_3')
('cle_4', ['valeur_4', 'valeur_5'])
('cle_5', 'valeur_6')

* Si la clé existe déjà l'ancienne valeur sera écrasée par la nouvelle valeur :
nom_du_dictionnaire["cle_3"] = " valeur_3_bis" for i in nom_du_dictionnaire.items() : print (i)

Retourne => ('cle_1', "valeur_1")
('cle_2', 'valeur_2')
('cle_3', 'valeur_3_bis')
('cle_4', ['valeur_4", "valeur_5'])
('cle_5', 'valeur_6')

Supprimer une paire clé-valeur :

* Avec le mot clé 'del' :
del nom_du_dictionnaire["cle_1"] print (nom_du_dictionnaire)

Retourne => {'cle_2': 'valeur_2',
'cle_3': 'valeur_3',
'cle_4': ['valeur_4", "valeur_5']}

vérifier l'existance d'une clé spécifique

On utilise le mot-clé 'in'.
Spécifié la clé recherché + in + nom de la variable du dictionnaire éxaminé. Le résultat renvoie un booléen qui indique si la clé est dans ce dictionnaire
"cle_2" in nom_du_dictionnaire

Retourne=> True

Autres methodes couramment utilisées pour manipuler les dictionnaires :

  • print ("keys() -> ​​Retourne une vue sur les clés du dictionnaire -> nom_du_dictionnaire.keys()")
  • print ("values() -> Retourne une vue sur les valeurs du dictionnaire -> nom_du_dictionnaire.values()")
  • print ("items() -> Retourne une vue sur les couples (clé, valeur) du dictionnaire -> nom_du_dictionnaire.items()")
  • print ("get(clé) -> Retourne la valeur associée à la clé spécifiée. Si la clé n'est pas présente dans le dictionnaire, retourne la valeur None -> nom_du_dictionnaire.get(clés)")
  • print ("pop(clé) -> Supprime la clé spécifiée et retourne la valeur associée. Si la clé n'est pas présente dans le dictionnaire, retourne la valeur None -> nom_du_dictionnaire.pop(clés)")
  • print ("clear() -> Supprime tous les éléments du dictionnaire -> nom_du_dictionnaire.clear()")

A savoir

Structure de données enregistrées dans des paires
clé - valeur. Les dictionnaires sont indiqués par des {}, il y a deux point (:) entre la clé et la valeur et une virgule (,) à la fin

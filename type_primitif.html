Utilisation

Les types de données, les 4 types de données primitifs : Les entiers (Integers) Les virgules flottantes (Float) Les chaines de caractères (String) Les booléen

Exemples

nombre = 2
phrase = ("Hello World")
booleen = True

retourne :
type (nombre) retourne "class 'int'"
type (phrase) retourne "class 'str'"
type (booleen)retourne "class 'bool'"

A savoir

Attention Python prend des points à la place des virgules Python déduit de la variable son type. La fonction type() determine le type de données d'une variable. Les types primitifs s'affichent uniquement dans la console

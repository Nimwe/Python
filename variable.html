A savoir

Les variables s'écrivent en snake case. Attentions les noms sont sensibles à la case.
Elles ne peuvent pas commencer par un chiffre. Ni accents, ni caractères spéciaux.

Elles sont constituées de 3 éléments :
Nom
Type (seront vus plus tard)
Valeur

Utilisation

Modifier une variable

Pour modifier une variable il suffit de lui assigner une nouvelle valeur

Exemple

livre = "Le seigneur des anneaux"
print (livre)
Livre = "Star War"
print (livre)

La concaténation

Concatenation de variables dans une chaine de caractéres avec f-string La chaine de caractères doit être précédée de f et les varables entre {}

Exemple Avec f-string

nom = "Dupont"
prenom = "Jean"
age = 30
print(f"Bonjour, je m'appelle {prenom} {nom} et j'ai {age} ans.)

Exemple Sans f-string, c'est plus long, plus complexe et il ne faut rien oublier en ponctuation.

nom = "Dupont"
prenom = "Jean"
age = 30
print("Bonjour, je m'appelle " + prenom +" "+ nom + " et j'ai" + " " + str(age) + " "+ "ans.")

A savoir

Elles sont créées avec [] et les éléments sont
séparés par , .
Ne pas oublier " " pour les strings. La première position de stockage dans une liste est : 0 et non pas 1.

Utilisation

Créer une liste stockée dans une variable :

nom_de_liste = ["element_1", "element_2", "element_3"]


Accéder aux éléments de la liste

nom_de_liste[N°element]

nom_de_liste[1]
Retourne 'element_1'

Dans Python on peut accéder aux éléments dans le sens inverse avec les négatives.

nom_de_liste[-1]
Retourne 'element_3'

On peut aussi l'utiliser pour les chaines de caractères

language = "Python"
language[2]
Retourne => "T"

Modifier - Remplacer

Utiliser l'indice de la place où la donnée doit être remplacée et y affecter une nouvelle valeur avec =

nom_de_liste[1] = "element_1_bis"
Print(nom_de_liste)
Retourne ["element_1_bis", "element_2", "element_3"]

Ajouter

Utiliser la methode -> .append()

nom_de_liste.append("element_4")
Print nom_de_liste
Retourne => ["element_1", "element_2", "element_3","element_4"]

Retirer

Utiliser la methode -> .remove()
Le .remove() retire seulement la 1ere instance du terme saisi

nom_de_liste.remove("element_4")
Print (nom_de_liste)
Retroune => ["element_1", "element_2", "element_3"]

Trier

Utiliser la méthode -> .sort()

nom_de_liste.sort'()
print(nom_de_liste) => ["element_a", "element_b", "element_c"]

Connaitre la longueur d'une liste

Utiliser len.
Attention : Tenter d'acceder à un index supérieur ou égal à la taille de la liste provoquera l'erreur -(IndexError)- car n'éxiste pas

len(nom_de_liste)
Retourne => 3

Autres methodes à utiliser avec les listes :

  • extend() -> ajoute plusieurs éléments à la fin de la liste
  • insert() -> insère un élément à une position donnée dans la liste
  • pop() -> supprime et renvoie lélément à une position donnée de la liste, ou
    le dernier élément si aucun indice n'est spécifié
  • index() -> renvoie à la 1ere occurence de l'élément spécifié
  • count() -> renvoie le nombre d'occurences de lélément spécifié dans la liste
  • reverse() -> inverse l'ordre des éléments de la liste

Les crochets

  • Pour créer une liste vide -> ma_liste = []
  • Pour créer des éléments
  • Pour acceder aux éléments via index
  • Modifier des éléments de la liste (Attention à ce que l'index existe sinon on aura IndexError)
  • Ajouter un élément avec .append()
  • Supprimer un élément avec Remove
  • Supprimer un élément via son index avec del

A noter que les chaines de caractères ont aussi des index sur lesquels il est possible de travailler en utillisant les mêmes méthodes

Les Tupples

Autres structure de liste utilisant les () plutôt que les []. La plupart des propriétés sont similaires aux listes normales, la différence est que les listes sont modifiables et les tupples immuables (Non modifiables après création).
